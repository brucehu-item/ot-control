如果你想在本地运行ot-control项目 请做如下准备:
 - 安装nodejs 18+, 推荐使用nvm来管理nodejs版本
 - 打开任意终端 
 - 执行cd ot-control
 - 执行npm install
 - 执行npm run dev

 下面是在终端执行npm run dev的输出，你就可以在浏览器中访问http://localhost:5173/ 来访问项目了。
